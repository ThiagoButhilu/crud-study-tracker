O Study Tracker é uma aplicação web desenvolvida exclusivamente para fins de portfólio, com o objetivo de demonstrar habilidades em desenvolvimento front-end, organização de estado e estruturação de interfaces.
O projeto simula um sistema de autenticação com CRUD de usuário, incluindo:
Criação de conta (cadastro)
Login e logout
Edição de dados do usuário
Exclusão de conta
Todo o fluxo de autenticação é feito sem backend, utilizando armazenamento local (como localStorage ou estado global), apenas para fins demonstrativos. Nenhum dado é persistido em servidor, e o sistema não deve ser utilizado em ambiente de produção.
Após o login, o usuário pode acessar uma área de controle de estudos, onde é possível organizar atividades, acompanhar o progresso e simular uma rotina de estudos personalizada.
Este projeto foi desenvolvido com foco em:
Boas práticas de front-end
Componentização
Gerenciamento de estado
Experiência do usuário (UX)
Clareza de código e organização de projeto
